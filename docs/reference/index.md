- Python 3.10

- Linux (all distributions)

## Changelog Highlights

### Version 0.1.0 (2025-01-01)

Initial release with:

- 24 feature extractors
- 387 ML-ready features
- Python API and CLI
- Kafka, Prometheus, PostgreSQL integrations
- Remote capture over WebSocket
- JA3/JA3S TLS fingerprinting
- HASSH SSH fingerprinting
- Tor/VPN/DoH detection

See [Changelog](changelog.md) for full details.
