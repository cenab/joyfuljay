## Stability

JoyfulJay provides stability guarantees through feature profiles:

| Profile | Stability | Breaking Changes |
|---------|-----------|------------------|
| JJ-CORE | Frozen | Major version bump |
| JJ-EXTENDED | Stable | Minor version bump |
| JJ-EXPERIMENTAL | Unstable | Any release |
