## Prerequisites

Before you begin, ensure you have:

- **Python 3.10 or later** installed
- **pip** package manager
- A **PCAP file** to analyze (or we'll show you how to capture one)

!!! tip "No PCAP file?"
    You can download sample PCAP files from [Wireshark's sample captures](https://wiki.wireshark.org/SampleCaptures) or use JoyfulJay's live capture feature.
